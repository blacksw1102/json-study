# json-study

JSON을 학습하기위한 repository입입니다.

## 학습 내용

- Maven으로 라이브러리 추가하는 방법

- JSONObject, JSONArray을 이용한 json 변환 방법

- Article 객체 데이터를 json형식으로 형변환하여 파일에 저장해놨다가, 다시 파일에서 json형식 데이터를 읽어와서 파싱을 한 후 Article 객체를 생성

- .gitignore 자동 생성법 : https://www.toptal.com/developers/gitignore

- .gitignore 재적용 방법 : git rm -r --cached .
